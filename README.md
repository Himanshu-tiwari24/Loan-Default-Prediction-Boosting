# 💰 Loan Default Prediction using Boosting Models

## 📌 Project Overview
This project focuses on predicting whether a loan applicant will default or not using advanced boosting algorithms. The goal is to build a robust machine learning model that helps financial institutions reduce risk and improve decision-making.

The project implements multiple boosting techniques and compares their performance to identify the most effective model.

---

## 🎯 Problem Statement
Loan default prediction is a critical task in the financial industry. Incorrect lending decisions can lead to significant financial losses. This project aims to develop a predictive system that can classify borrowers as likely to default or not based on historical data.

---

## 📊 Dataset
- Structured financial dataset
- Includes borrower details such as income, loan amount, credit history, etc.
- Preprocessing steps applied:
  - Handling missing values
  - Encoding categorical variables
  - Feature scaling (if required)

---

## 🧠 Models Used
The following Boosting algorithms were implemented:

- 🔹 AdaBoost
- 🔹 Gradient Boosting
- 🔹 XGBoost
- 🔹 CatBoost
- 🔹 LightGBM

Each model was trained and evaluated using appropriate performance metrics.

---

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC Score

Model comparison was performed to select the best-performing boosting algorithm.
